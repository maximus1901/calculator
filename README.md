# Calculator (Rust)

A small command-line calculator written in Rust.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Clear, fast CLI workflow
- Safe error handling (no panics for expected user input)
- Extensible design (easy to add more operators / parsing)

## Getting started

### Prerequisites
- Rust toolchain (stable) + Cargo

### Build
```bash
cargo build
